SASS vs SCSS

SCSS uses standard css formatting, with curly brackets.

SASS uses it's own formatting, which relies on indentations rather than curly brackets.

Getting started with SASS

  1. First, make sure that you have Ruby installed by navigating to your home directory (cd ~), and typing "ruby -v". If Ruby is not installed, follow the directions for installing Ruby
  2. Second, make sure you have Sass installed by by navigating to your home directory (cd ~), and typing "sass -v".
